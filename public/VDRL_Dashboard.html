How to use this dashboard
What this is
It tracks every vendor document through the review loop Vendor → Kent (EPCM) → ADNOC → Kent → Vendor. Each number, chart and table reflects the documents currently in view; everything is cross-linked, so any filter updates the whole page.
Filtering
  • Slicers (Equipment, Vendor, Discipline, Stage, Status, Revision, Asset): tick any boxes — counts beside each option show how many docs match, and impossible options grey out. Multiple ticks in one slicer = OR; different slicers combine with AND.
  • Stage group buttons (With Vendor / Kent / ADNOC) narrow to who holds the document now.
  • Master search (top bar) scans every field. Type compressor AND siemens for all terms, or vs1 OR pump for any.
  • Active filters show as chips — click the ✕ on a chip, or Reset all, to clear.
Charts
Click any bar/slice to filter. The Filter / Highlight toggle decides whether a click narrows the data (Filter) or just spotlights it while keeping context (Highlight). Interactions (gear) lets you pin a chart so it ignores filters and always shows totals.
Current Stage matrix
Pending counts by stage and vendor. Click a row to filter; click ▸ to collapse a group; drag its left edge to resize.
Review-cycle analysis
Turnaround built from the four transition dates. Avg full cycle = days from vendor submission to return to vendor (median & P90 too). Kent hold / ADNOC hold show who holds documents longest (the bottleneck). The table lists the slowest completed documents — click a row for full detail.
Document register
  • Sort: click a header. Resize: drag a header's right edge. Reorder: drag a header left/right, or use ↑↓ in the Columns menu.
  • Columns button: show/hide any of the 25 source columns plus calculated ones (Equipment, Revision-calc, Days, cycle days). Also export to CSV.
  • Row click opens the full document detail (drill-through) with every field.
Saved views & settings
Use + Save current view to store a filter + layout combo and recall it later. Your column choices, widths, order, and collapsed panels are remembered automatically for your profile.
Data & updates
Data loads automatically from the server folder; Refresh latest reloads it. When a new version is published you'll see a "Reload" bar at the top. Admins get an Admin button to approve users and manage access.
Tip: hover a calculated value or truncated cell to see the full text. Calculated fields: Equipment Name, Stage group, Revision (Calc), Days in stage, and the cycle metrics above.
