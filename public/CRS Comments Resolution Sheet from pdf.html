Step 3: Post-Processing Macro

Converts floating images to Excel cell pictures.

Sub ConvertImagesToCells() ' Loops through all pictures in Column B (Row 12+) ' and converts them to "Picture in Cell" Dim ws As Worksheet Dim shp As Shape Dim targetRange As Range Dim i As Long Set ws = ActiveSheet ' Disable screen updating for speed Application.ScreenUpdating = False ' Loop BACKWARDS because Cutting changes the shape count For i = ws.Shapes.Count To 1 Step -1 Set shp = ws.Shapes(i) ' Check if it is a Picture If shp.Type = msoPicture Or shp.Type = msoLinkedPicture Then ' Check if it is in Column B (Col 2) and Row >= 12 If shp.TopLeftCell.Column = 2 And shp.TopLeftCell.Row >= 12 Then Set targetRange = shp.TopLeftCell ' Cut the floating picture shp.Cut ' Select the cell and Paste as Picture-in-Cell targetRange.Select On Error Resume Next Selection.PastePictureInCell On Error GoTo 0 End If End If Next i Application.ScreenUpdating = True MsgBox "All images in Column B converted to Cells!", vbInformation End Sub
