Developer Guide

How to add new apps and persist data in App Hub.

1

Create Your File

Duplicate template.html to start a new page. This file already contains the necessary connections to the App Core.

cp template.html my_new_tool.html
2

Connect to Data

Inside your new file, locate the script section at the bottom. Use the Core object to save data.

For Standard HTML/JS:

// Listen for login
Core.onLogin = function(user) {
    // Load data
    Core.db.collection('users').doc(user.uid).collection('app_data').doc('my_tool_id').get()
        .then(doc => {
            if(doc.exists) console.log("Loaded:", doc.data());
        });
};

// Save data
function saveData(data) {
    Core.db.collection('users').doc(Core.user.uid).collection('app_data').doc('my_tool_id')
        .set(data, { merge: true });
}

For React Apps:

See tva_tool.html for a complete example using useEffect hooks to sync state automatically.

3

Register in Dashboard

Once your file is ready and uploaded:

  1. Go to the Dashboard (index.html).
  2. Click App Manager in the top right.
  3. Enter the name, icon, and filename (e.g., my_new_tool.html).
  4. Click Add App.
