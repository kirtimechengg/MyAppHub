MRT / AGES checklist

API 617 Part 1 cl. 6.3.5 mechanical running test, plus ADNOC AGES amendments. Status, comments and edits are stored with the project.

Open point — AGES 6.3.5.11 is self-conflicting. One paragraph requires shaft seals removed for inspection after a successful MRT; the next restricts removal to oil-film seals only, excluding mechanical-contact and dry gas seals. Confirm with COMPANY which seals get pulled on this machine and record it in the test agenda.

User guide

How to run an ASME PTC-10 Type 2 performance-test validation, start to finish. New users can follow the steps in order.

What this tool does

It validates a shop performance test of a centrifugal compressor run on a substitute gas (an ASME PTC-10 Type 2 test), against the contract guarantee per API 617. It sizes the required test gas and speed, checks that the test stays inside the code's allowable similitude limits (volume ratio, flow coefficient, Machine Mach and Reynolds numbers), compares corrected/as-measured performance to the guarantee, carries the API 617 + ADNOC AGES mechanical-running-test checklist, and produces a printable A4 report. Every calculated value shows the formula with your numbers substituted — nothing is hidden.

Step-by-step workflow

1Units & settingsPick Metric (default) or US customary. Switching offers to convert values you've already entered. Optionally set pressure / temperature / flow display units. The three unit-dependent constants (gas constant, temperature offset, power constant) update automatically.
2Specified conditionsEnter the guaranteed operating point (blue cells). Choose how you supply the discharge side — discharge capacity Q₂ or volume ratio — and how you supply power — absorbed power (incl. mechanical losses) or gas power + losses. Efficiency uses gas power; the ±4% guarantee check uses absorbed power. Derived values appear on the right with their formulas. The guarantee-vs-test comparison table sits below.
3Test gas selectionPick a gas from the dropdown and an ambient temperature (20/25/30/35 °C) to auto-fill MW, k, Z₁ and test inlet temperature. For a mixture, choose "Gas mixture" and enter properties manually. The screening table flags gases whose MW exceeds the specified gas (needed to keep test speed under MCS) and picks the closest k. Required test pressure ratio, head and speed are computed with full formula traces.
4Verify — with / without RemEnter the shop test readings. "With correction" applies vendor Reynolds-correction factors to head, efficiency, work and flow coefficients; "without" compares as-measured data directly (conservative). Similitude deviations are checked against PTC-10 limits, and the Mach / Reynolds points are plotted live on the Fig. 3.3 and 3.5 charts beside the original PTC-10 snaps.
5MRT / AGES checklistWork through the mechanical-running-test check points (API 617 cl. 6.3.5 + ADNOC AGES). Set Y / N / N/A, add comments, edit text, add or delete rows, or import/export as JSON or CSV. Status, comments and edits are saved with the project.
6AttachmentsDrag-drop images, paste screenshots (Ctrl + V), or browse. Images merge into the report PDF, one per page. PDF files are indexed on an appendix page (they can't be embedded — screenshot them or merge afterwards).
7Report of testSet manufacturer / witness / date and the correction basis, then "Open full A4 report". Print → Save as PDF (Edge: margins None). The report includes cover, all calculations with formulas, the comparison table, both verification programs, the allowable-departure charts with snaps, the scorecard with the ACCEPTED / NOT ACCEPTED determination, the checklist, and attachments.

Saving your work

Save project writes a .t2p.json file holding every input, the unit system, the mode selections, the full checklist (statuses and comments), and attachments. Open restores it exactly. There is no auto-save — nothing is written to disk until you click Save, which avoids corporate policy issues. Keep the file on a network drive or email it to move a project between machines.

Installing as a Windows app

Open Type2Tool.html in Microsoft Edge → ⋯ menu → Apps → Install this site as an app. It gets its own window, a Start-menu entry and a taskbar icon, and runs fully offline with no admin rights. To update, replace the HTML file. Use the ☰ button (top-left) to collapse the side menu for more screen space.

Important limitations

  • The gas-property presets are representative screening values at low test pressure. For a contractual calculation, confirm MW, k and Z₁ from a proper equation of state (e.g. NIST REFPROP) for your exact gas, pressure and temperature — especially for CO₂, R-134a and mixtures.
  • The Fig. 3.3 / 3.5 curves are digitised reproductions for working use, shown beside the original snaps for cross-checking. They are not a substitute for the purchased ASME PTC-10 code text.
  • Machine Mach and Reynolds checks assume the design Reynolds number is at or above the 90,000 Type-2 minimum; the tool flags it when it isn't.
  • This tool supports engineering judgement — it does not replace the governing standards or the reviewing engineer's sign-off.
